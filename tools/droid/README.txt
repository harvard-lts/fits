The DROID XML signature file with a "_Alt" suffix indicates the original DROID Signature file has been modified from its original.
The V102 file has two minor modifications that
give priority of "Plain Text File", ".txt" files over "TRIM", ".txt" files and
give priority of "Plain Text File", ".txt" files over "ESRI ArcInfo Coverage Annotation File", ".txt" files and
give priority of "MD5 File", ".md5" files over "FrameMD5", ".md5" files.
